# Guia de Contribuição - LunAI

Obrigado por considerar contribuir para o LunAI! 🎉

## Como Contribuir

### Reportando Bugs

1. Verifique se o bug já foi reportado em [Issues](../../issues)
2. Se não encontrou, abra uma nova issue com:
   - Título claro e descritivo
   - Descrição detalhada do comportamento esperado e observado
   - Passos para reproduzir
   - Screenshots (se aplicável)
   - Seu ambiente (OS, versão Python, etc)

### Sugerindo Melhorias

1. Abra uma nova issue com o label `enhancement`
2. Descreva a melhoria de forma clara
3. Explique por que seria útil

### Pull Requests

1. Fork o repositório
2. Crie uma branch para sua feature (`git checkout -b feature/AmazingFeature`)
3. Commit suas mudanças (`git commit -m 'Add some AmazingFeature'`)
4. Push para a branch (`git push origin feature/AmazingFeature`)
5. Abra um Pull Request

## Padrões de Código

- Use Python 3.8+
- Siga PEP 8
- Adicione docstrings em funções
- Escreva testes para novas features
- Use type hints

## Processo de Review

Todos os PRs serão revisados antes de serem mergeados. Mantemos a qualidade do código em primeiro lugar.
