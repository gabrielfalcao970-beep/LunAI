# LunAI - Changelog

## [1.0.0] - 2024-01-16

### Added
- ✨ Initial release of LunAI
- 💬 Conversational chatbot with natural language processing
- 😊 Sentiment analysis with emotion detection
- 🎨 10 dynamic avatar expressions
- 📝 Automatic chat history saving
- ⚙️ Configuration system with multiple options
- 🎨 PyQt5 GUI with dark/light themes
- 📦 Local data storage with JSON support
- 📱 API integration framework (OpenAI, Google Gemini)
- 🐧 Linux build and installation scripts
- 📦 Standalone executable creation with PyInstaller
- 🧠 Memory management for conversation context
- 🔒 Privacy-first (runs locally, no cloud required)
- 📚 Comprehensive documentation

### Features
- Offline mode (no API required)
- Customizable personality and tone
- Widget-style GUI that stays on top
- Chat history with timestamps
- Emotion-based responses
- Expandable response system
- Multi-language ready (Portuguese, English)
- Automatic dependency installation

### Technical
- Python 3.8+
- PyQt5 for GUI
- TextBlob for sentiment analysis
- Support for OpenAI GPT and Google Gemini APIs
- Modular architecture for easy extension
